qual a linguagem de programacao usada no nosso backend?
JavaScript

o que e node?
É runtime para rodar o Javascript e criar API's

o q e NPM?
É um gerenciador de pacotes para o NODE, adicionar e remover pacotes, como tambem fazer outras modificacoes no projeto (NODE Packege Manager)

o q é o express?
É um framework para o Node que facilita a criacao de servidores e API's

o que e uma API?
É uma interface de programacao de aplicativos, onde o desenvolvedor pode criar a consumir servicos e dados de outros sistemas

o que e interface?
É uma forma como o usuario interage com o sistema, pode ser visual ou de comando, onde o usuariopode enviar comandos e receber respostas do sistema

como mudar o formato de importacao de require? (padrao commonjs) para module?
Precisa alterar o packege.jason e colocar "type": "module" no arquivo

o que http?
É um protocolo de comunicacao entre cliente e servidor, onde o cliente faz uma requisicao e o servidor responde

quais o metodos de requisicao http e para que serve cada um?
GET = para buscar dados
POST = para criar dados
PUT = para atualizar dados
DELETE = para deletar dados
PATCH = para atualizar parcialmente os dados

para que serve o --watch ao rodar o server?
Ele fica observando os arquivos e rodar o servidor em modo de desenvolvimento, onde ele vai reiniciar automaticamente quando houver alguma mudanca no codigo

design patterns?
Design Patterns (Padrões de Projeto) são soluções reutilizáveis e generalistas para problemas recorrentes no desenvolvimento de software. Eles não são códigos prontos, mas sim modelos (receitas) de alto nível para estruturar classes e objetos, facilitando a manutenção, escalabilidade e o entendimento do código por equipes. São divididos em criacionais, estruturais e comportamentais.

-acesso aos dados
